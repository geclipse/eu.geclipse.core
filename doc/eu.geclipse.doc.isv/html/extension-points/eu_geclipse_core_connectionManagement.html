connectionManagement

Identifier:
eu.geclipse.core.connectionManagement

Since:
This extension point is part of an interim API that is still under development and expected to change significantly before reaching stability. It is being made available at this early stage to solicit feedback from pioneering adopters on the understanding that any code that uses this API will almost certainly be broken (repeatedly) as the API evolves.

Description:
This extension point is for providing File Systems (extension of org.eclipse.core.filesystem.filesystems) that could be used by gExplorer. Each contributor can also provide Connection Wizard Pages (extension of eu.geclipse.ui.connection.connectionWizard if some special initiation of File System is required). In other words: each connection implementation must extend this extension point (e.g gridFTP connection). This is representation of transport layer.

Configuration Markup:

<!ELEMENT extension (connectionProtocol)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #REQUIRED

name  CDATA #IMPLIED>

This is standard Eclipse's element used for extension point management within Eclipse's Registry



<!ELEMENT connectionProtocol EMPTY>

<!ATTLIST connectionProtocol

id       CDATA #REQUIRED

name     CDATA #REQUIRED

class    CDATA #REQUIRED

wizardID CDATA #REQUIRED>

  • id - Value of this attribute is used to associate File System provided by eu.geclipse.core.connectionManagement extension point with Connection Wizard Pages provided by eu.geclipse.ui.connection.connectionWizard extension point (it must be used in eu.geclipse.ui.connection.connectionWizard)
  • name - Value of name attribute will be used to present connection parameters to user, therefore it should be user readable name.
  • class - Value of this attribute binds connectionManagement with FileSystem. It should be the same as value of attribute class in element run (in element filesystem) in org.eclipse.core.filesystem.filesystems extension point.
  • wizardID - Set to true if this extension point requires data from user to connect. This data is taken from Connection Wizard Pages. Developer should provide Connection Wizard Pages by extending eu.geclipse.ui.connection.connectionWizard extension point.
