Factory, which produces detailed items of submitted job for "Job Details" view


  • id -
  • class - Class, which implements this factory
  • sourceJobClass - Class extending IGridJob, for which this factory can produce detailed items. This factory will be used only for IGridJob-s, which are instance of specified sourceJobClass
  • sourceJobStatusClass - Optionally class extending IGridJobStatus, for which this factory can produce detailed items. If you specify it, then this factory will be used only for IGridJob-s, for which getStatus() return object which is instance of sourceJobStatusClass.
