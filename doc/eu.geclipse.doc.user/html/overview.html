g-Eclipse

Access Grid and Cloud infrastructures with Eclipse

g-Eclipse User Guide

The g-Eclipse project provides an integrated workbench framework to access the power of existing Grid and Cloud infrastructures. The framework contains (views, editors, wizards, etc.) to customize Grid users' applications, to manage Grid resources and to support the development cycle of new Grid applications.

The g-Eclipse framework provide a middleware independent Grid model and implementation for different Grid and Cloud middlewares. The user guide explains the generic concepts as well as middleware specific features.

The g-Eclipse project was supported by the European Commission under contract number 034327.

More information about the project can be found at www.eclipse.org/geclipse. Additional information about more middleware implementations can be found at www.geclipse.eu.
