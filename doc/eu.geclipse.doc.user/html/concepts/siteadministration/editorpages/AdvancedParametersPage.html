g-Eclipse - QDL Editor - Advanced Parameters Page

The Advanced Parameters page provides an interface for manipulating extended Batch Queue parameters. It consists of one section: the Advanced Queue Parameters section.

Advanced Queue Parameters

The Priority is mandatory and defines the priority of the Batch Queue. The Queue Name is the Unique Identified for each Batch Queue. Priority is of type "Integer".

The Max Running Jobs field is optional and specifies the maximum number of jobs in the Batch Queue allowed to run at any given time. Max. Running Jobs is of type "Integer".

The Max Jobs in Queue field is optional and specifies the maximum number of jobs allowed in the Batch Queue at any given time (includes idle, running, and blocked jobs). Max. Jobs in Queue is of type "Integer".

The Resources field is optional and specifies specifies the cumulative resources available to all jobs running in the queue.. Max. Jobs in Queue is of type "Integer".
