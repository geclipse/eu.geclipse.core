g-Eclipse - Download CA certificates

Open the g-Eclipse > CA Certificates preference page. If you have no CA Certificates installed so far, you should see a preference window similar to figure 1.


Figure 1: CA Certificates Preference Window

To add certificates from a repository, click on Add from Repository and enter the URL of the repository in the box that opens up. By default, there will be a repository location already provided for you, which you may just accept and click OK. The various certificates will be downloaded (figure 2), unzipped and installed.


Figure 2: Downloading CA Certificates

To add the certifcates from a local directory, click on Add from Directory, choose the directory containing the certificates and click OK.
The various certificates found will then be copied to a dedicated directory in your workspace. The resulting view will be similar to figure 3.


Figure 3: CA Certificates appear in the Preference Window

Finally click OK to complete the step of downloading CA certificates.
