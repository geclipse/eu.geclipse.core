glogin - A Multifunctional, Interactive Tunnel into the Grid

glogin offers bidirectional secure connections between clients and grid nodes and is currently based on the Globus Toolkit. It includes standard shell functionality as well as TCP forwarding and is used by numerous components in g-Eclipse.

Additionally to the java based version already included in g-Eclipse, a native executable version installed on the desired remote node is required. The glogin source code can be acquired from http://www.gup.uni-linz.ac.at/glogin and the generated executable should be placed in the directory search path for executables files.

The following components make use of glogin:

  • Launching and Debugging Applications using the Launch Configurations
  • Terminal
  • GVid
