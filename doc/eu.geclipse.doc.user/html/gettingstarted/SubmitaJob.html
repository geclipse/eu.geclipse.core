g-Eclipse - Submit a Job

Now that you have a job description file in JSDL, you can submit it as a job to the Grid. Right-click on the jsdl file you created in the previous step and select Submit Job.... A dialog opens up where you can choose from a number of Workload Management Servers (WMS) to submit your job to. Pick a WMS server and click Finish to submit the job. You can see your submitted job appear in the Job folder in your Grid Project.

From the main menu, choose the Window > Show View > Other > g-Eclipse > Jobs View to see the current status of your submitted job.
