g-Eclipse - Getting started

When you start the g-Eclipse framework, you will see the following Eclipse window.

On the top left you see the Grid Project View with the Grid Resource View below it. In the lower part, additional Grid access tools are available such as the Grid Terminal, Grid WebView, gExplorer, GVid, etc. You can add additional view by choosing them from Window > Show View > Other > g-Eclipse.

At startup, the editor is empty. Depending on the action you start, you will get the corresponding editor in the Editor area.

Before you can start interacting with the Grid, you need to set up your g-Eclipse environment. The following configuration is neccessary:

  • Download the certificates of the approved Certificate Authorities.
  • Locate your Grid Certificate to be able to generate Grid Authentication Tokens.
  • Declare the Virtual Organisations, in which you are a member.
  • Create a new Grid Project.
  • Create a new Grid Job.
