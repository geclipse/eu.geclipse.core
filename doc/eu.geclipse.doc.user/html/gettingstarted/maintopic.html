g-Eclipse - Getting started

When you start the g-Eclipse framework, you will see an Eclipse window similar to the following:

You can open the g-Eclipse User Perspective by choosing it from Window > Open Perspective > Other > g-Eclipse (User). The Eclipse Workbench will now look similar to the following:

In the top left, you see the Grid Project View with the Grid Resource View below it. In the lower part, additional Grid access views are available such as the Connections, Jobs and Authentication Tokens Views. To the right is the Properties View. You can add additional views by choosing them from Window > Show View > Other > g-Eclipse.

At startup, the central part of the workbench, the editor, is empty. Depending on the action you start, you will get the corresponding editor in the Editor area.

Before you can start interacting with the Grid, you need to set up your g-Eclipse environment. The following configuration is necessary:

  • Download the certificates of the approved Certificate Authorities.
  • Declare the Virtual Organisation(s), of which you are a member.
  • Locate your Grid Certificate and generate Grid Authentication Tokens.
  • Create a new Grid Project.
  • Create a new Grid Job.
  • Submit a Grid Job.
  • Application Deployment.
