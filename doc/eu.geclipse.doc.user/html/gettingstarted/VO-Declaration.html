Virtual Organisation Declaration

Go to Window > Preferences > g-Eclipse > VO-Declaration.

Click on the Add button and, in the dialog that opens, put in the VO specific settings such as Name, Host, Port, Host DN, and Information System Endpoint. The path to the server's certificate should also be specified; the URL could point to a remote or a local file. Click Finish.




In the dialog that opens up, select the VO that you just created and click OK to complete the VO declaration process.
