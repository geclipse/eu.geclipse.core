g-Eclipse - Create a new Grid Project

After your personal preferences have been completed in the previous step and you have provided g-Eclipse with information on how you want to authenticate yourself on the Grid, you are ready to create your first Grid Project.

Your Desktop might look similar to the image in figure 1. Once you complete the steps described here the leftmost pane (see figure 6) will contain a tree-view of your created project called the Grid Project view.


Figure 1: A screenshot of the empty g-Eclipse perspective

"Right Click" into that view and select New > Grid Project to launch the New Grid Project Wizard, which is shown in figure 2. This wizard will help you create your first g-Eclipse project.


Figure 2: The New Grid Project Wizard

On the first page of the New Grid-Project Wizard, specify the name you want your grid project to be called and the location where you want to store its files on your local harddisk. For the location you may accept the default settings. Then click the Next button to get to the following page, shown in figure 3.



Figure 3: The New Grid Project Wizard

This page asks you to choose your Virtual Organization (VO). Select your VO from the list and click the Next button. If you have no VO available in the list, you can create a new one by clicking on the New VO button, selecting the type of VO and clicking Next.



Figure 4: Attributes for creating a new Virtual Organization

As figure 4 shows, for a new VO you are asked to provide the following information: the Name , Host, Port, Host DN, Information System Endpoint, and the VO server URL.

Clicking the Finish Button will take you back to the VO specification page where you can select from a list of available VO's. At this point you should see something similar to figure 3.

Click Finish and your newly created project will appear in the "Grid Projects" View as shown in figure 6.


Figure 6: New Grid Project tree in the panel view

It will contain three subfolders

  • Filesystems - this directory will contain filesystems you will use from your grid jobs
  • Job Descriptions - this directory will contain descriptions of your jobs, which can be created with the New Job Wizard
  • Jobs - this directory will contain information about your past and present jobs running on the Grid
