g-Eclipse - Access Grid infrastructures with Eclipse

This guide provides instructions for using the g-Eclipse framework in the Eclipse workbench.

The g-Eclipse project aims to build an integrated workbench framework to access the power of existing Grid infrastructures. The framework will provide tools to customize Grid users' applications, to manage Grid resources and to support the development cycle of new Grid applications.

The g-Eclipse project is supported by the European Commission under contract number 034327.

More information about the project can be found at www.eclipse.org/geclipse.
