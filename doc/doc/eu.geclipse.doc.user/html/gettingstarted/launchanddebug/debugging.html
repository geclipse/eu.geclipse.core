Debugging an Application on the Grid

After setting up a launch configuration of one of the following types,

  • Java Grid Application
  • C/C++ Grid Application
  • C/C++ MPI Grid Application

debugging can be started and the application will be listed in the debug view.



From here you can debug your application as if it were running locally. In case of an MPI application, all processes are listed and can be controlled separately. To set process-specific breakpoints you can use filtering in the breakpoint properties.



It is possible to interact with the application via the console.
